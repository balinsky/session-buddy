Import from CSV

Upload a CSV file with your tunes. New tunes will be added to your existing collection.

Expected columns: Name, Type, Key, Parts, Incipit A, Incipit B, Incipit C, Count, Added, Where, Who, Mnemonic, Tunebooks, Date Learned, Favorite, Thesession ID, Setting, Notes, Composer, Last Practiced Date, Instrument, Sequence ID

Learning status: Include a Learned column — X imports as Memorized, anything else imports as Not Learned. The "Learning" status is set manually in the app.
